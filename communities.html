Comunidades

Python Alicante

Comunidad para gente que le gusta y quiere aprender a programar en Python. Estamos en Alicante, ¡Pero todo el mundo es bienvenido!

Python Almería

Comunidad local de Almería. Donde se trata temas relacionados con el lenguaje de programación Python. Organizamos charlas y talleres sobre python

Python Barcelona & PyLadies Barcelona

Python Barcelona (PyBCN) is a registered non-profit association. Our objectives are to spread the use and knowledge of the Python programming language and to be a meeting point for all the Barcelona Python community. PyLadies BCN is an international mentorship group with a focus on helping more women become active participants and leaders in the Python open-source community. Our mission is to promote, educate and advance a diverse Python community through outreach, education, conferences, events and social gatherings

Python Canarias

Python Canarias es una asociación sin ánimo de lucro cuyos objetivos, son, entre otros, promover el uso del lenguaje de programación Python, dar soporte a quienes deseen aprenderlo y promover la programación como una competencia educativa básica

Python Castilla y León & PyData Salamanca

PyData Salamanca: Grupo abierto e interdisciplinar orientado al análisis de datos y su difusión en la ciudad de Salamanca. Python CyL: Nurevo grupo que trata de aunar a toda las personas entusiastas del lengiaje Python en Castilla y León
